About Our Project

Our project involves developing a movie title generator using artificial neural networks and the CMU Movie Summary Corpus dataset. The goal is to take a movie summary and generate an appropriate title. We chose this theme because it relates to Natural Language Processing (NLP) and data processing, while utilizing machine learning techniques to accomplish the task. We are working with a dataset of 42,000 movies, containing their summaries, titles, and Wikipedia IDs.

The generator is integrated into a web application built with FastAPI, where users can submit a movie summary and receive a generated title in return.
